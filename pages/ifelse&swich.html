Javascript Learning If else and swich

conditional statements are used to decide the execution flow, based on different conditions. When the condition is true, specific action is performed, and if the condition is false, another action will perform

Condition Meaning
a==b a is equal to b
a != b a is not equal to b
a < b a is less than b
a > b a is greater than b
a <= b a is less than or equal to b
a >= b a is greater than or equal to b
a===b a has the same value and same type as b

1. If Condition

if(condition expression) { 
        // code to be executed 
        }else{
            //Execute this code.
            }
        //
        let salary = 500;
        if( salary > 1000)
        {
        console.log("My Salary is greater than 1000");
        }else{
        console.log("My Salary is less than or equal to 1000");
        }
        // Output:-My Salary is less than or equal to 1000
